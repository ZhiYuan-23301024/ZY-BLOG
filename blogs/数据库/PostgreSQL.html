    PostgreSQL

    ZhiYuan 12/6/2025

    用户询问如何用命令行连接PostgreSQL数据库，我需要查看项目中的数据库配置文件，了解当前的数据库配置信息，然后提供相应的命令行连接指导。

    toolName: view_files

    status: success

    filePath: e:\go-project\AICare\configs\DBconfig.yaml

    我需要查看项目中的数据库连接相关代码，了解如何在Go代码中连接PostgreSQL数据库，以便为用户提供更全面的指导。

    toolName: view_files

    status: success

    filePath: e:\go-project\AICare\pkg\config\database.go

    现在我已经了解了项目的数据库配置和连接代码，接下来我需要回答用户的问题：如何用命令行连接PostgreSQL数据库。我将基于DBconfig.yaml中的配置信息，提供使用psql命令行工具连接PostgreSQL数据库的具体步骤，包括安装方法和连接命令。要使用命令行连接PostgreSQL数据库，您需要使用psql工具，这是PostgreSQL自带的命令行客户端。以下是连接步骤：

    # 1. 安装PostgreSQL（如果尚未安装）

    如果您的系统上还没有安装PostgreSQL，可以从官方网站下载并安装：

    • PostgreSQL官方下载页面 (opens new window)

    在安装过程中，确保勾选"Command Line Tools"选项，以便安装psql工具。

    # 2. 确认psql已安装

    打开命令提示符（cmd）或PowerShell，运行以下命令确认psql是否已安装：

    psql --version

    # 3. 连接到PostgreSQL数据库

    根据您项目中DBconfig.yaml的配置，使用以下命令连接数据库：

    psql -h localhost -p 5432 -U postgres -d aicare

    参数说明：

    • -h localhost: 数据库服务器地址（与配置文件中的host一致）
    • -p 5432: 数据库端口（与配置文件中的port一致）
    • -U postgres: 用户名（与配置文件中的user一致）
    • -d aicare: 数据库名称（与配置文件中的dbname一致）

    # 4. 输入密码

    如果您的PostgreSQL用户设置了密码（配置文件中password字段），系统会提示您输入密码。根据您的配置文件，密码目前为空，直接按Enter键即可。

    # 5. 成功连接后的操作

    连接成功后，您将看到类似以下的提示符：

    aicare=#

    此时您可以执行SQL命令，例如：

    • 查看所有表：\dt
    • 执行SQL文件：\i scripts/createTable.sql
    • 退出连接：\q

    # 6. 使用完整DSN连接（可选）

    您也可以使用完整的DSN（数据源名称）来连接数据库：

    psql "host=localhost port=5432 user=postgres password='' dbname=aicare sslmode=disable"

    这与您项目中DBconfig.yaml文件中的dsn字段格式一致。

    # 7. 解决常见问题

    • "psql: command not found"：确保PostgreSQL的bin目录已添加到系统环境变量PATH中
    • 连接超时：检查PostgreSQL服务是否正在运行，端口是否正确
    • 认证失败：确认用户名和密码是否正确，以及PostgreSQL的pg_hba.conf文件是否允许该用户连接

    # 8. 在项目中使用

    您的项目已经通过pkg/config/database.go中的InitDatabase函数实现了Go代码与PostgreSQL的连接，该函数会根据DBconfig.yaml中的配置自动构建DSN并建立连接池。

    如果您需要修改数据库连接配置，只需更新DBconfig.yaml文件中的相应字段即可。
