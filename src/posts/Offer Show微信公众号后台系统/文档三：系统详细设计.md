
#### 数据结构设计

本部分定义系统核心实体及其字段，确保数据标准化与可扩展性。

**薪资信息实体**

|字段名|类型|必填|说明|示例值|
|---|---|---|---|---|
|id|String|是|唯一标识符（UUID）|“sal_001”|
|company_name|String|是|公司全称|“腾讯科技”|
|position_name|String|是|岗位名称|“后端开发工程师”|
|location|String|是|公司所在地|“北京”|
|salary_description|String|是|薪资描述（文本）|“月薪25k，16薪”|
|salary_min|Decimal|否|年薪下限（万元）|30.0|
|salary_max|Decimal|否|年薪上限（万元）|40.0|
|offer_type|Enum|是|Offer类型：校招/实习/社招|“校招”|
|education|String|是|学历要求|“硕士”|
|industry|String|是|所属行业|“互联网”|
|remarks|String|否|其他备注|“有房补”|
|credibility|Integer|是|可信度评分（1-5）|4|
|create_time|DateTime|是|发布时间|“2025-03-20 14:30:00”|
|likes|Integer|是|点赞数|120|

**企业校招信息实体**

|字段名|类型|必填|说明|示例值|
|---|---|---|---|---|
|id|String|是|唯一标识符|“job_001”|
|company_name|String|是|公司全称|“阿里巴巴”|
|position_name|String|是|岗位名称|“产品经理”|
|location|String|是|工作地点|“杭州”|
|industry|String|是|所属行业|“互联网”|
|education_requirement|String|是|学历要求|“本科及以上”|
|apply_deadline|DateTime|是|网申截止时间|“2025-04-30 23:59:59”|
|apply_link|String|是|投递链接|“[https://example.com/apply”](https://example.com/apply%E2%80%9D)|
|create_time|DateTime|是|发布时间|“2025-03-18 10:00:00”|

#### API接口设计

所有接口遵循RESTful风格，返回格式统一为JSON。标准方法包含：`GET`（查询）、`POST`（创建）、`PUT`（更新）、`DELETE`（删除）、`PATCH`（部分更新）、`replace`（全量替换，通过`PUT`实现）。

**企业校招信息查询**

- **功能描述**：根据检索条件分页查询企业校招岗位信息。
    
- **接口地址**：`GET /api/v1/companies/jobs`
    
- **请求参数**：

| 参数名          | 类型      | 必填  | 说明         | 示例值   |
| ------------ | ------- | --- | ---------- | ----- |
| location     | String  | 否   | 工作地点       | “北京”  |
| company_name | String  | 否   | 公司名称（模糊查询） | “腾讯”  |
| industry     | String  | 否   | 所属行业       | “互联网” |
| page         | Integer | 是   | 页码（从1开始）   | 1     |
| page_size    | Integer | 是   | 每页数量（最大50） | 20    |
    
- **响应示例**：
    

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "total": 150,
    "page": 1,
    "page_size": 20,
    "items": [
      {
        "id": "job_001",
        "company_name": "腾讯科技",
        "position_name": "后端开发工程师",
        "location": "北京",
        "industry": "互联网",
        "education_requirement": "硕士及以上",
        "apply_deadline": "2025-04-30 23:59:59",
        "apply_link": "https://example.com/apply"
      }
    ]
  }
}
```

**薪资爆料发布**

- **功能描述**：用户提交薪资爆料信息。
- **接口地址**：`POST /api/v1/salaries`
- **请求体**：

```json
{
  "company_name": "字节跳动",
  "position_name": "前端开发工程师",
  "location": "北京",
  "salary_description": "月薪28k，15薪",
  "salary_min": 35.0,
  "salary_max": 45.0,
  "offer_type": "校招",
  "education": "硕士",
  "industry": "互联网",
  "remarks": "有年终奖",
  "credibility": 5
}
```

- **响应示例**：

```json
{
  "code": 201,
  "message": "创建成功",
  "data": {
    "id": "sal_001",
    "create_time": "2025-03-20 14:30:00"
  }
}
```

**薪资信息更新**

- **功能描述**：更新指定薪资信息（全量替换或部分更新）。
- **接口地址**：
    - 全量更新：`PUT /api/v1/salaries/{id}`
    - 部分更新：`PATCH /api/v1/salaries/{id}`
- **请求体**（以部分更新为例）：

```json
{
  "salary_description": "月薪30k，16薪",
  "credibility": 5
}
```

- **响应示例**：

```json
{
  "code": 200,
  "message": "更新成功",
  "data": {
    "id": "sal_001",
    "update_time": "2025-03-21 10:00:00"
  }
}
```

**薪资信息删除**

- **功能描述**：删除指定薪资信息。
- **接口地址**：`DELETE /api/v1/salaries/{id}`
- **响应示例**：

